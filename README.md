# IL Assembly Project Template

Template for compiling raw .NET IL code.

Includes `primespeed.il` (prime number finder) as example.

Note: This won't be faster than C#. For learning only.

## Usage

1. Install .NET
2. Clone repo
3. Run `dotnet build` or `dotnet publish`
4. Run executable from `bin/Release/net10.0/{platform}/publish/`
```bash
git clone https://github.com/mg0x7BE/ilasm-project-template.git
cd ilasm-project-template
dotnet build
```
